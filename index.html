Household ID

1
Create a free Firebase project

Go to console.firebase.google.com → Add project. Any name works. Disable Google Analytics if asked — not needed.

2
Add a Web App & copy the config

Click the </> Web icon → register with any nickname. After registering, Firebase shows a firebaseConfig block. Copy the whole { … } object and paste it here:

Invalid config — paste the full { ... } object from Firebase.
3
Enable Firestore Database

In Firebase Console → Build → Firestore Database → Create database. Choose Start in test mode, pick any region, click Enable.

4
Set your Household ID

This private ID links all your devices. Generate one, or type your own. To sync a second device, enter the exact same ID there.
