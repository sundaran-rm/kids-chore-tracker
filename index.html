❓ How to Use Chore Quest
🧒 Managing Kids
➕
Add a kid
Click + Add Kid in the header. Each kid gets their own chores, tokens, streak, and rewards.
🎨
Switch theme
Click the coloured dot next to a kid's name tab to pick Galaxy, Dino, Princess, Ocean, or Unicorn theme.
✏️
Rename a kid
Click directly on the name in the tab and type a new one. Changes save automatically.
☀️ Daily Tasks & Chores
➕
Add a chore
Click + Add in any section header. A new chore appears and the edit modal opens — set the name, tokens, difficulty, schedule, and instructions.
📝
Chore instructions (hover)
If a chore has instructions set in its edit modal, hovering over the chore card reveals them in a strip below the title. Useful for reminding kids exactly what's expected.
🟡
Difficulty badge
Click the coloured badge (🟢 Easy / 🟡 Med / 🔴 Hard) on any chore to cycle difficulty. Set it to None only via the ✏️ edit modal. Difficulty is hidden from kids in Kids Mode.
📅
Set a schedule
Open the ✏️ edit modal and tap the day buttons (S M T W T F S) to choose which days a chore appears. Greyed-out chores are not scheduled for today.
✔️
Tick off a chore
Click the circle button on the right of any chore. Tokens are added automatically. Use ↩ to undo.
⭐
Chore of the Week
Click the ⭐ button on a chore to mark it as Chore of the Week — it earns 2× tokens for the whole week. The gold border highlights which chore it is.
⚡
Bonus chores
One-off or extra chores that don't follow a daily schedule. Great for special tasks or extra effort.
✨
Custom sections
Click + Section in the header to create named groups (e.g. "Homework", "Pet Care"). Drag chores between sections.
📋
Templates
Click the Templates button in the Daily Tasks header to load a ready-made pack. Save your own packs with the 💾 Save button.
⭐ Tokens & Rewards
⭐
Earning tokens
Tokens are earned automatically when chores are ticked off. The gold coin in the right column shows the running total.
🎁
Rewards
Add rewards (e.g. "Movie Night – 50★") in the Rewards section. Click Redeem to spend tokens when a reward is earned.
🛍
Reward Shop
Click 🛍 Shop in the Rewards header to browse suggested rewards. Customise shop items and costs per kid via Settings → Reward Shop Items.
🎯
Goals
Set a savings goal in the Wallet section — link it to a reward to show a progress bar towards something specific.
✏️
Manual adjustments
Use the + Add / − Remove buttons in the Wallet to give bonus tokens or make corrections. Add a note to explain why.
💪 XP & Levels
📈
XP progress bar
The purple bar just below the header shows the current XP level and progress to the next one. XP is earned by completing chores and reduced by undo. Spending tokens on rewards does not reduce XP.
🌱
Levels
There are 6 levels: 🌱 Rookie → ⭐ Helper → 🌟 Star → 🏆 Champion → 👑 Legend → 🔮 Master. Hover the XP bar to see exact XP count.
🔥 Streaks & Achievements
🔥
Streaks
A streak increases each day at least one chore is completed. Miss a day and it resets. The record is saved separately. The 30-day streak chart is in the Earned section.
🏅
Achievements & Badges
Hover any badge to see what it requires and — if earned — the exact date it was unlocked. Thresholds shown reflect any custom edits you've made.
⚙️
Customise achievements
Go to Settings → Achievements & Badges to change how many chores/streak days/tokens are needed for each default badge, and create entirely new achievements. Settings are per kid. Raising a threshold will un-earn badges the kid no longer qualifies for.

🔒 Kids Mode
🔒
Turning it on
Click Kids Mode in the header (or Settings). All editing controls hide — kids can only view, tick chores, and see their instructions by hovering.
🔑
Parent PIN
Set a 4-digit PIN in Settings → Parent PIN. Once set, Kids Mode can only be unlocked with the PIN. The PIN syncs across all devices.
🔓
Unlocking
Click the green Kids Mode 🔒 badge in the header, enter the PIN, and parent controls return.
☑ Settings
🏅
Achievements & Badges
Edit thresholds for numeric badges, and create custom achievements based on total chores, streak days, or tokens earned. Per kid.
🛍
Reward Shop Items
Override the suggested token cost for any default shop item, and add your own custom items. Changes apply per kid.
☁ Cloud Sync
☁
Setting up Firebase
Go to Settings → Cloud Database and follow the 4-step guide to connect a free Firebase project. Takes about 5 minutes.
📱
Linking devices
Use the same Household ID on every device. All chores, tokens, badges, and templates sync in real time.
🟢
Sync status
The dot in the header shows sync state — green = synced, yellow = saving, orange = offline pending, red = error.
Tip: Press Escape to collapse or expand all sections at once. Drag the ☰ handle on any chore or reward to reorder it. Hover a badge to see its requirement and earned date.
