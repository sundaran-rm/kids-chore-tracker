❓ How to Use Chore Quest
🧒 Managing Kids
➕
Add a kid
Tap the kid picker (top bar) → + Add Kid. Each kid gets their own chores, tokens, streak, and rewards.
🎨
Switch theme
Tap the kid picker → tap the coloured dot next to a name to pick Galaxy, Dino, Princess, Ocean, or Unicorn.
✏️
Rename a kid
Tap the kid picker, then tap the name text and type a new one. Saves automatically.
☀️ Chores Tab
✔️
Tick off a chore
Tap the circle button on any chore to mark it done. Tokens are added instantly. Tap again to undo.
➕
Add a chore
Tap + Add in any section. Type a name and set token value with the ★ badge.
📅
Scheduling
Tap the day buttons (S M T W T F S) under a chore to set which days it appears. Greyed = not today.
📋
Templates
Tap the 📋 button in the Daily Tasks header to load a ready-made chore pack. Save your own with 💾 Save.
✨
Custom sections
Tap ➕ in the header to add a named chore group (e.g. "Homework"). Drag ☰ to reorder.
🎁 Rewards Tab
🎁
Add rewards
Tap + Add in the Rewards section. Set a name and token cost. Tap Redeem to spend tokens.
💰 Wallet Tab
⭐
Token balance
The gold coin shows total tokens. Earn them by ticking chores; spend them by redeeming rewards.
✏️
Manual adjustments
Tap + Add or − Remove to give bonus tokens or make corrections. Add a note for the history.
🎯
Goals
Set a savings goal and link it to a reward to show a progress bar towards something the kid is working towards.
📊 Stats Tab
🔥
Streaks
Increases each day at least one chore is completed. Miss a day and it resets to zero.
🏅
Achievements & Badges
14 badges to unlock. Greyed out = still locked. Earned badges show the date they were unlocked.

🔒 Kids Mode & PIN
🔒
Enable Kids Mode
Go to Settings → Kids Mode. Editing controls hide — kids can only view and tick chores.
🔑
Set a PIN
Settings → Parent PIN. A 4-digit PIN protects Kids Mode from being turned off. Syncs across all devices.
🔓
Unlock
Tap the green Kids badge in the top bar, enter your PIN, and parent controls return.
☁ Cloud Sync
☁
Connect Firebase
Settings → Cloud Database. Follow the 4-step guide to link a free Firebase project (~5 mins).
📱
Link devices
Use the same Household ID on every device. Chores, tokens, templates, and PIN sync in real time.
🟢
Sync status
The dot icon in the top bar shows status: green = synced, yellow = saving, orange = offline, red = error.
Tip: Drag the ☰ handle on any chore or reward to reorder it. Tap a section header to collapse it and save screen space.
📋 Chore Templates

Choose a pack to fill your chore list. You can edit everything after adding.
